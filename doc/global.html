logicaCalibracion(puntero)

Esta funcion sirve como callback una vez que el detector de marcadores, haya detectado un marcador. A su vez, dentro de la misma esta la lógica de la etapa de Calibracion. La etapa de calibración es un proceso donde a partir de un orden de colores, debes de seleccionar cada color, dependiendo de el orden indicado. La misma funcion identifica si ya se detectaron todos los elementos de prueba, inicia el nivel de Memorama

Parameters:
Name Type Description
puntero THREE.Object3D Es el objeto que la instancia de DetectorAR, traspuso la posición del marcador
